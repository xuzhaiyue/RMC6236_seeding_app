# RMC-6236 Seeding Calculator

Streamlit app for calculating cell suspension, medium, total cells, total volume, and optional RMC-6236 stock volume for time-course seeding.

## Local run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Cloud

Deploy this repository with:

- Main file path: `app.py`
- Python dependencies: `requirements.txt`
